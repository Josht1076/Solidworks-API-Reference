Remarks

If PageIndex is the same index as returned by ISWPropertySheet::AddActivePage, then ControlDispatch will contain the Dispatch pointer of the ActiveX control.

 

If developing a C++ application, use swPropertySheetCreateControlNotify to register for this notification.
