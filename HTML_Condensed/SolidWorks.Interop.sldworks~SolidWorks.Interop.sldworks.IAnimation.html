This interface is:

  • obsolete and has not been superseded.

  • nonfunctional in SOLIDWORKS 2008 and later.

Use the interfaces related to motion studies introduced in SOLIDWORKS 2008 to access animation and simulation.

Remarks

Do not confuse the Animation Controller pop-up toolbar available in the core SOLIDWORKS software with SOLIDWORKS Animator, a SOLIDWORKS add-in. SOLIDWORKS Animator provides more functionality than the Animation Controller pop-up toolbar. See the SOLIDWORKS Animator online Help for details.
