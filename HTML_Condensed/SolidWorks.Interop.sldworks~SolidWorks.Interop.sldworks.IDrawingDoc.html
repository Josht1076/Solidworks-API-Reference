Remarks

Creating, aligning, and accessing drawing views are a few of the functions you find on the IDrawingDoc object.

The SOLIDWORKS API also includes functions that are common to all document types. For example, determining the file name associated with a document is a common operation. The SOLIDWORKS API uses the IModelDoc2 object to expose common document-level functions.

Events are implemented with delegates in the Microsoft .NET Framework. See the Overview topic for a list of delegates for this interface.
