Remarks

To get the IMate2 object, you must first get it as a IFeature object and then use IFeature::GetSpecificFeature2 or QueryInterface to return the IMate2 interface.

The IMate2 object is actually a subfeature beneath a MateGroup feature. To access the IMate2 subfeature by traversing the FeatureManager design tree, you can call IFeature::GetFirstSubFeature or IFeature::IGetFirstSubFeature and then IFeature::GetNextSubFeature or IFeature::IGetNextSubFeature when the current feature type is MateGroup.
