Remarks

Read Pattern Features and their Feature Data Objects.

Before calling IFeatureManager::CreateDefinition, you can use IModelDocExtension::SelectByID2 to pre-select the entities needed to create the circular pattern feature.

To select the required entities, use these selection Marks:

  • features = 4

  • points = 32

  • sketches = 64

  • faces = 128

  • bodies = 256

Or, after calling IFeatureManger::CreateDefinition, you can explicitly set the corresponding properties on this feature data object.

Either way, you must call IFeatureManager::CreateFeature after initializing this feature data object in order to successfully create a sketch-driven pattern.

For more information, see the Sketch Driven Patterns topic in the SOLIDWORKS Help.
