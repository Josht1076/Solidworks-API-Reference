Remarks

A SketchSegment can represent a sketch arc, line, ellipse, parabola or spline.

ISketchSegment provides functions that are generic to every type of sketch segment. For example, every sketch segment has an ID and can be programmatically selected. Therefore, the ISketchSegment interface provides functions to obtain the ID and to select the item. (For access to sketched points, see the ISketchPoint object)
