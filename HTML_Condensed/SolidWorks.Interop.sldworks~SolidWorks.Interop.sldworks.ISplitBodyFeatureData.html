Remarks

IFeature::GetTypeName2 and IFeature::GetTypeName return:

  • Split for a feature that was created by splitting a part into multiple parts by using either IFeatureManager::PostSplitBody or the Split feature in the user interface. You can access a split-body feature's data using the ISplitBodyFeatureData interface.
  • SplitBody for a body created by splitting a part and saving the body to a part. You cannot access the data of a split body saved to a part.
