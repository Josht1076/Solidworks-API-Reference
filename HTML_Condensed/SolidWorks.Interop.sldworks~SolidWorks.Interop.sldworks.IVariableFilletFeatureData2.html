Remarks

To create a variable fillet feature, follow the instructions in the Remarks section of IFeatureManager::FeatureFillet3.

To edit a variable fillet feature:

  1. Call IFeature::GetDefinition.
  2. Call IVariableFilletFeatureData2::AccessSelections.
  3. Modify properties on this interface as needed and call IFeature::ModifyDefinition.
  4. If nothing is modified, call IVariableFilletFeatureData2::ReleaseSelectionAccess.

For more information, read the Variable Size Fillets topic in the SOLIDWORKS user-interface help.
