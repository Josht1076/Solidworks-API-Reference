Remarks

This interface supports SOLIDWORKS Model Based Definition (MBD), which is an integrated drawingless manufacturing solution for SOLIDWORKS. Use ISldWorks::LoadAddIn to load the MBD add-in.

See the SOLIDWORKS Help for details about MBD.
